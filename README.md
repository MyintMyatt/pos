# POS (Point Of Sale)

## Features

- Admin Section
- User Management
- Menu Management
- Inventory Management
- Report

## Tech Stack 

- Java 17, Spring Boot 3.5.6
- Spring Vault
- Open Api Documentation
- Redis
- PostgreSQL
- Docker

## Clone Repo
```bash
  git clone https://github.com/MyintMyatt/pos-backend.git
```
1. **Change project dir**
```bash
  cd pos-backend
```
2. **Run docker compose**
```bash
   docker compose up -d
```

3. Api Documentation
```bash
   http://localhost:8080/api-doc
```
